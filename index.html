Getting Started #back to top

All communications with the bot is on Facebook Messenger, the prefix for the commands is !. You can access to the bot here. The communication is pretty simple with these basic commands.
We will see here the 3 basic commands, !help, !about and !ping

To call a command it's simple, juste send your command ( with the prefix ! to the bot and he will answer you with the output.

Help command

The !help command is the the basic command, the output of this command is basicly all the commands and how to use them. The help output is the default output if you send an invalid command or word.


As you see the command show like this :
The first and last line is a separator to clearly see the output, the second and third lines are keys to understand the usage of the commands and after a blank line, all the other lines are commands with usage.

For the example let's take the !ping command, there are no required argument to the command so you can send the command to the bot without other informations and this command will output a nice pong ! to you !

About command

The !about command is a basic information command, it's output some of informations ( developper, contact mail, host ). The usage is simple, there is no required arguments so you can just send the command to the bot, he will be glad to answer !

Ping command

The !ping command is pretty useless, it has an utility only if you want to test if the bot is online, if so he will answer you by a cool pong !

Registring

The process of registring is pretty easy, only one command is required to launch the procedure, the !subscribe command. If you remember the help section, you know that some commands can have required argument, this is one of them, let's see the command in the help section :

In the usage of this command we see two word beetween "[]" this are required arguments, the first argument is the SCC of the officer receive the reports, so if you followed the alert, you. The second argument is the name of your chapter like USS Somevessel please, do not include the regio only "USS" and the name. Use ARS with the website #back to top

You can access to the website client for Automatic Report System here this website handle all the functions of a client, you can edit and save your report in the user place.

Main Page

Navigation Bar

The main page is minimalist, the navigation bar have some usefull links, the Member Place link is the user page ( or the login page if you are not login ). the Main link to the main page and the CV Generator ! link to the generator of SFI CV.

Body

The main page body contains 3 cards which open a full page of explainations, what is ARS ?, How ARS works ? and the FAQ. A link of contact is included in the bottom of the page
